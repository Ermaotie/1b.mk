爬取微博并将最新一条推送至微信

发表于 2020-05-21 更新于 2020-08-06

首先感谢dataabc所写的weibo-scrawler项目，非常感谢能够开源。

  • 部署
  • 调参
    • config.json
    • msg.py
  • 使用

部署

clone 项目

git clone https://github.com/Ermaotie/weibo-crawler.git

安装相关module

pip install -r requirement.txt

调参

config.json

详细设置见github
这里必须将文件格式保存为json
若想要定期执行，请将since_date设置为 int 型整数

msg.py

  1. 方糖server酱绑定微信并获取SCKEY
  2. 填入SCKEY，修改文件路径为 
weibo/微博用户名/微博id.json

使用

使用crontab 定期执行

crontab -e
# 编辑定时执行任务
0 12 * * * python weibo.py # 绝对路径
0 12 * * * python msg.py # 绝对路径
# 每天12时0分 执行xxx
# 注意python版本

crontab 命令format

*    *    *    *    *    your command
-    -    -    -    -
|    |    |    |    |
|    |    |    |    +----- 星期中星期几 (0 - 7) (星期天 为0)
|    |    |    +---------- 月份 (1 - 12) 
|    |    +--------------- 一个月中的第几天 (1 - 31)
|    +-------------------- 小时 (0 - 23)
+------------------------- 分钟 (0 - 59)

查看任务列表

crontab -l
