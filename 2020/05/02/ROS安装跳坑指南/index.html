ROS安装跳坑指南

发表于 2020-05-02 更新于 2020-08-06

官方Kinetic版安装链接：http://wiki.ros.org/cn/kinetic/Installation/Ubuntu

国内安装真的是太不友好了！！！！！！

常见问题：

1.安装包下载太慢

更换清华镜像源及软件源
镜像源列表：
https://mirror.tuna.tsinghua.edu.cn/help/ubuntu/

重写地址： 

/etc/apt/sources.list

软件源: 

sudo sh -c '. /etc/lsb-release && echo "deb http://mirrors.tuna.tsinghua.edu.cn/ros/ubuntu/ $DISTRIB_CODENAME main" > /etc/apt/sources.list.d/ros-latest.list'

2.ros版本不一致
查看方法：

rosdep --version
apt-cache policy python-rosdep

版本若不同：
需卸载通过pip安装的版本，通过apt安装

sudo apt-get remove --purge python-rosdep python-pip
sudo apt-get update
sudo apt-get install python-rosdep

3.所有的timeout
S：
①更改NS服务
所在目录：/etc/resolv.conf
推荐更换为谷歌DNS

nameserver 8.8.8.8
nameserver 8.8.4.4

需要注意的是，ns设置将在每次重新启动后刷新

②直接将域名所指向的IP设为静态(推荐)
首先到ipaddress.com查找需要设置为静态的域名

raw.githubusercontent.com
github.com
github.global.ssl.fastly.net
codeload.github.com

以ip在前，域名在后的形式写入

/etc/hosts

例：

199.232.68.133    raw.githubusercontent.com

(安装完成后记得将新写入的静态解析擦除，因为GITHUB服务器的IP会更换）

③使用代理服务
通过shadowsocks或者V2ray，在ubuntu上可能会更加难以操控，如果你了解如何使用，那么这个办法是最省事的，请到v2ray主页查看如何使用。当然也有一个不太成熟的类似V2rayN式GUI界面的软件——v2rayL(需要设置http以及socket端口转发）

④安装在哪里？
不推荐安装在虚拟机以及树莓派中，但依然是可以安装的。

为什么？
运行前需要编译很多相关文件

Ubuntu 16是最适合装Kinetic的版本
关于树莓派3/3+/4安装：
直接烧好已经安装好ROS Kinetic的镜像：点击进入
