    • 换源

      换源

      npm config set registry https://registry.npm.taobao.org

使用Hexo

Hexo概述

Hexo 是一款可以通过git工具，发布静态博文的一款优秀博客系统。它最重要的特点就是轻便，轻且便

Hexo官方文档

安装Hexo

# 全局安装
npm install -g hexo-cli

使用Hexo

初始化Hexo文件夹

控制台中打开所要初始化的文件夹

hexo init

常用命令

新建xxx

hexo new demoTitle
hexo new page "demoPage"
hexo new tag "demoTag"
hexo new category "demoName"

维护使用

# 清除原有生成的静态博客
hexo clean
# 生成静态博客
hexo g  # hexo generate
# 部署（到github）
hexo d  # hexo deploy
# 本地查看 http://localhost:4000
hexo s  # hexo server
